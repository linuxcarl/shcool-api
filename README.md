## Installation

Installation

```

npm install

```

Run test

```

npm run test

```

Develop

If you want run mode develop, run the next line:

```

npm run dev

```

Compile app to js

If you want run mode production, first compile app, run the next line:

```

tsc

```

Production

If you want run mode production, run the next line:

```

npm run start

```
